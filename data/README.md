# Host Aggregation Data

This directory contains the generated CSV files from the host aggregation process.

## Files

- **`host_entries_YYYYMMDD_HHMMSS.csv`** - Timestamped datasets from each run
- **`latest.csv`** - The most recent dataset (always current)

## Access

These files are automatically generated by the GitHub Actions workflow and can be accessed via:

```
https://raw.githubusercontent.com/hostdni/host-aggregator/hosts/data/latest.csv
```

## Data Format

Each CSV file contains the following columns:
- `entry` - The hostname/domain to block
- `category` - Category of the blocked content (Adware & Malware, Fake news, etc.)
- `action` - Always "block"
- `description` - Description of the blocked content
- `risk` - Risk level (currently empty)
- `is_enabled` - Always "True"

## Updates

The data is updated daily at 2:00 AM UTC via automated GitHub Actions workflow.
